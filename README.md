# IA & Santé au Travail — version multipage

Cette version remplace le routeur par fragments par de vraies pages statiques. Le dossier peut être publié tel quel sur GitHub Pages avec un domaine personnalisé.

## Publication

- Déposer l’ensemble du contenu à la racine du dépôt.
- Dans GitHub : **Settings > Pages > Deploy from a branch**, puis sélectionner `main` et `/root`.
- Conserver le fichier `CNAME` si le domaine personnalisé est déjà configuré.

## Newsletter

Le formulaire fonctionne en mode de préinscription par messagerie tant que `assets/js/config.js` ne contient pas l’URL du backend. Le dossier `newsletter-backend` fournit un exemple Google Apps Script avec confirmation par courriel.
